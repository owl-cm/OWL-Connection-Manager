Documentation

Welcome to the official documentation for OWL Connection Manager. OWL is a premium, high-performance tool designed for power users who manage remote infrastructure. This guide covers everything from basic setup to advanced features.

Pro Tip

Use Ctrl + Shift + P within the application to open the Command Palette for quick access to all features.

Installation

OWL is currently optimized for Linux environments. You can install it using the Debian package or run it from source.

Debian Package (.deb)

  1. Download the latest .deb file from the Home Page.
  2. Install it using your package manager: 
    sudo dpkg -i owl-connection-manager_*.deb
sudo apt-get install -f

From Source

git clone https://github.com/owl-cm/OWL-Connection-Manager.git
cd OWL-Connection-Manager
npm install
npm start

Connection Management

OWL provides a powerful tree-based system for organizing your connections. You can group servers into folders, drag and drop them to reorder, and use color labels for quick identification.

Smart Folders

Create nested folders to mirror your infrastructure. For example, you can have a "Production" folder with subfolders for "Web Servers" and "Database Servers".

Connection Labels

Assign colors to connections to visually distinguish between environments:

  • Red: Production
  • Yellow: Staging
  • Green: Development

Smart Cloning

Quickly duplicate a connection configuration. The "Smart Clone" feature automatically places the new connection in the same folder as the original.

SSH & Remote Protocols

The heart of OWL is its high-performance terminal emulator and multi-protocol support.

Supported Protocols

  • SSH: Full-featured terminal with xterm.js support.
  • RDP: Connect to Windows machines with integrated Remote Desktop support.
  • VNC: Access graphical Linux or macOS sessions via VNC.

Terminal Features

  • Full Color Support: Supports 256-color and true-color themes.
  • Mouse Events: Interact with terminal applications like htop or vim using your mouse.
  • Automated Login: OWL intelligently handles password prompts and SSH key passphrases.
  • Host Fingerprints: Automatically handles "Are you sure you want to continue connecting?" prompts for new hosts.

Broadcast Mode

One of OWL's most powerful features. When enabled, any command you type in the active terminal is simultaneously sent to all other open terminal sessions. This is ideal for updating multiple servers at once.

Bastion / Jump Hosts

Connect securely to servers behind firewalls using integrated ProxyJump support. Simply configure the Bastion details in the connection modal.

SFTP File Explorer

Manage remote files without leaving the application. The integrated SFTP explorer provides a graphical interface for your remote filesystem.

Capabilities

  • Drag-and-Drop: Upload files by dragging them from your local machine into the explorer.
  • Directory Insights: Instantly calculate the size of remote directories with a single click.
  • File Operations: Rename, delete, and download files with ease.
  • Breadcrumb Navigation: Quickly jump between parent directories.

Real-Time Monitoring

Keep an eye on your server's health with live metrics and process management.

Metrics Dashboard

View live graphs and values for:

  • CPU Usage: Real-time percentage of processor load.
  • RAM Usage: Monitor memory consumption.
  • Disk Usage: Track available space on the root partition.

Remote Process Manager

A full-featured task manager for your remote server. Search for processes, view their resource usage, and terminate them directly from the UI.

Security

Security is not an afterthought in OWL; it's the foundation. We use industry-standard encryption to ensure your data stays private.

Secure Vault

All sensitive data, including passwords, private key paths, and passphrases, is stored in a local vault. This vault is encrypted using AES-256-GCM.

Master Password

When you first launch OWL, you'll be asked to set a Master Password. This password is used to derive the encryption key for your vault using PBKDF2 with 100,000 iterations. OWL never stores your master password.

Auto-Lock Timer

For enhanced security, you can configure an auto-lock timer. If the application is idle for a specified period, it will automatically lock the vault and require your master password to resume.

Advanced Tools

Command Palette & Snippets

Access any feature or snippet by pressing Ctrl + Shift + P. The palette is searchable and designed for a keyboard-centric workflow.

Command Snippets

Save frequently used commands as snippets for one-click execution. Snippets can be managed directly from the Command Palette or the dedicated Snippets manager.

  • Quick Execution: Run snippets on any active terminal session.
  • Organization: Categorize snippets for different tasks (e.g., Docker, Git, System Maintenance).

Session Logging

Automatically log all terminal output for auditing or debugging. OWL includes a built-in log viewer with advanced filtering:

  • Timeframe Filters: View logs from today, this week, or a custom date range.
  • In-Log Search: Find specific text within a log file instantly.
  • Log Rotation: Automatically rotate log files when they reach a certain size (e.g., 100MB) to save disk space.

Backups & Migration

Export your entire configuration to an encrypted backup file. This makes it easy to move your connections to a new machine securely.
